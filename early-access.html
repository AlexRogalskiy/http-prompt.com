Why isn't it free?

HTTP Prompt started out as a open source software. But open source ≠ free. To sustain ourselves, we employ this "pay for early access" business model with a subscription plan for users who want to live on the bleeding edge.

How does it work?

Say we have a new major release v2, we don't release it to the public immediately. Instead, we release it to the paid users first. Not only that the paid users are given an access key to download the latest package (v2) from our private server, they are also entitled to better support. Meanwhile, free users can only use the open source version (v1). Once we have a newer release v3, we push v2 to the public as open source, and release v3 to the paid users.

So what's new in v0.10.x?

HTTP Prompt v0.10.x is probably the most exciting release since we released Unix-like pipelines feature! In this version, we introduce OpenAPI specification (fka Swagger) support. You'll be able to specify an OpenAPI specification with --spec command line option then use ls and cd commands to navigate between API endpoints with autocomplete like a charm.
